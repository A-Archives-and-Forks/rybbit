# Shopify

Rybbit can be integrated with your Shopify store to capture events, track conversions, and analyze user behavior.

## Installation

1. **Edit Theme**

  - In Shopify, go to **Online Store > Themes > Actions > Edit Code**.

2. **Insert Rybbit Script**

  - Open `theme.liquid` in the **Layout** folder.
  - Just before the closing `</head>` tag, insert the following:

  ```html
  <script src="YOUR_INSTANCE_ADDRESS" async data-site-id="YOUR_SITE_ID"></script>
  ```

3. **Save Changes**

  - Save the file to apply the changes to your live store.

## Example Use Cases

After integrating Rybbit, you can:

- Identify popular products that are not converting
- Understand which items are often purchased together
- Measure marketing campaign impact on sales
- Analyze regional purchasing behavior
