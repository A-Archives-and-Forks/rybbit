# Framer

Rybbit enables you to gather analytics on user behavior, capture custom events, record sessions, implement feature flags, and more within your Framer site.

## Installation

1. **Retrieve Your Web Snippet:**

  - Navigate to your Rybbit project settings.
  - Copy the provided web snippet, which resembles the following:

  ```html
  <script src="YOUR_INSTANCE_ADDRESS" async data-site-id="YOUR_SITE_ID"></script>
  ```

2. **Add the Snippet to Your Framer Project:**

  - Open your Framer project.
  - Click the gear icon in the top-right corner to access your site settings.
  - If you haven't already, upgrade to the "Mini" site plan to enable custom code injection.
  - Insert the copied JavaScript snippet into the **Custom Code** section, ensuring it's placed within the `<head>` tag.
